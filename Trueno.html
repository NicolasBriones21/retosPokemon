Tutorial para hacer papitas fritas

  1. Primero necesitas comprar las papas
  2. Lavar las papas
  3. Pelar las papas (opcional)
  4. Cortar las papas en bastones no tan delgados
  5. Hervir las papas a temperatura media baja
  6. Secar muy bien las papas para evitar problemas con el aceite
  7. En un sarten agregar un poco de aceite y calentarlo
  8. Echar las papas en el sarten con mucho cuidado
  9. Dejar que se frian las papas
  10. Dejar las papitas sobre una servilleta para escurrir el exceso de aceite
  11. Agregrar sal al gusto
  12. Disfrutar
